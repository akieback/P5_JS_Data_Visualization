Always check the Data!

These 2 maps have the purpose to show that the same dataset can have 2 completely different results. The left graph shows the number of SSI's, while the right graph shows the number of SSI's in relation to the number of operations performed.
Even though both are based on true data, the result is completely different. So the left graph of Marin County could be used to show that few SSI's occurred as opposed to Los Angeles County, for example. However, if you look at the right graph, you can see that Los Angeles still has a lower SSI rate than Marin.
Therefore, it is important that any data visualization is done in a way that conveys information correctly.
